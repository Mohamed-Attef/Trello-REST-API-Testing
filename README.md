# Trello REST API Testing Project

## Overview

This project demonstrates API testing of Trello REST APIs using Postman. The objective was to validate the functionality, reliability, and correctness of various Trello operations through API requests and automated test scripts.

## Scope

The project covers testing of core Trello features, including:

* Board Management

  * Create Board
  * Get Board
  * Update Board
  * Delete Board

* List Management

  * Create List
  * Get List
  * Update List
  * Archive List
  * Unarchive List

* Card Management

  * Create Card
  * Get Card
  * Update Card
  * Delete Card

* Checklist Management

  * Create Checklist
  * Get Checklist
  * Update Checklist
  * Delete Checklist

## Testing Activities

* Performed API testing using Postman.
* Validated HTTP status codes and response payloads.
* Verified response fields and data integrity.
* Implemented automated assertions using Postman test scripts.
* Tested CRUD operations using GET, POST, PUT, and DELETE methods.
* Used environment and collection variables for dynamic test execution.
* Verified business rules and endpoint behavior.
